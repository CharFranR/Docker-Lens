Docker-Lens

Accede a tu base de datos desde el terminal.
Sin configuración, sin complicaciones.

$ pip install dlens-py
GitHub PyPI v1.0.5

Un comando. Todas las bases.

Docker-Lens detecta automáticamente qué DB tienes en tu docker-compose.yml. No necesitas configurar nada.

Detección automática

Analiza tu docker-compose.yml y encuentra la DB en segundos. PostgreSQL, MySQL, MariaDB, MongoDB, SQLite.

Queries desde la terminal

Ejecuta SQL o MongoDB queries directamente desde la CLI. Sin abrir otro cliente, sin cambiar de ventana.

Exportación

Exporta tablas a CSV o migrá toda la base a SQLite con un solo comando. Ideal para backups y análisis.

Schema inspection

Ve columnas, tipos y constraints de cualquier tabla. Conoce tu estructura sin abrir el cliente de la DB.

Shell interactivo

Abre una sesión interactiva con tu DB directamente desde Docker-Lens. psql, mysql, mongosh — el que necesites.

Rust core

El motor de detección y conexión está escrito en Rust con PyO3. Rápido, sin dependencias del sistema, bundleado.
